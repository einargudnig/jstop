**Key decisions for this plan:**
- MenuBarExtra style: `.window` — displays a SwiftUI popover (not a native menu list). Required because Phase 2 needs interactive buttons (kill, open in browser) which don't work in `.menu` style.
- Icon: `systemImage: "terminal"` — recognizable, no custom asset needed for v1
- PopoverView fixed width: 320pt — standard for menu bar utility apps
- App termination: expose a Quit button since the app has no Dock icon/menu bar quit option visible to the user

struct NodeProcessesApp: App {

        } label: {

        }
        .menuBarExtraStyle(.window)

Notes:
- `.menuBarExtraStyle(.window)` creates a popover-style window (NOT a native menu). This is required for Phase 2's interactive kill/open buttons.
- `Image(systemName: "terminal")` uses SF Symbols — no asset required.
- Do NOT add a `WindowGroup` scene. A MenuBarExtra-only app with LSUIElement=YES has no Dock icon and no visible window outside the popover.

Create `NodeProcesses/ProcessListView.swift`:

```swift
import SwiftUI

struct ProcessListView: View {
    var body: some View {
        VStack(alignment: .leading, spacing: 0) {
            // Header
            HStack {
                Text("Node Processes")
                    .font(.headline)
                Spacer()
            }
            .padding(.horizontal, 12)
            .padding(.vertical, 8)

            Divider()

            // Placeholder content — replaced in Phase 2
            VStack(spacing: 6) {
                Image(systemName: "terminal")
                    .font(.system(size: 28))
                    .foregroundColor(.secondary)
                Text("No processes running")
                    .foregroundColor(.secondary)
                    .font(.callout)
            }
            .frame(maxWidth: .infinity)
            .padding(.vertical, 24)

            Divider()

            // Quit button — necessary since app has no Dock icon
            Button("Quit Node Processes") {
                NSApplication.shared.terminate(nil)
            }
            .buttonStyle(.plain)
            .foregroundColor(.secondary)
            .padding(.horizontal, 12)
            .padding(.vertical, 8)
        }
        .frame(width: 320)
    }
}
```

Notes:
- Fixed width 320pt — standard for menu bar utility popovers.
- Quit button is essential: with LSUIElement=YES and no WindowGroup, users have no standard way to quit.
- Placeholder terminal icon + "No processes running" text will be replaced in Phase 2.
